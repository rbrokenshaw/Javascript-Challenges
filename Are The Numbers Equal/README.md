<h1>Are the Numbers Equal?</h1>

<p>Create a function that takes two integers and checks if they are equal.</p>

<h2>Examples</h2>

<ul>
	<li>isEqual(5, 6) ➞ false</li>
	<li>isEqual(1, 1) ➞ true</li>
	<li>isEqual("1", 1) ➞ false</li>
</ul>
