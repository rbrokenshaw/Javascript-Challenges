<h1>Find the Index (Part 1)</h1>

<p>Create a function that finds the index of a given item.</p>

<h2>Examples</h2>
<ul>
	<li>search([1, 5, 3], 5) ➞ 1</li>
	<li>search([9, 8, 3], 3) ➞ 2</li>
	<li>search([1, 2, 3], 4) ➞ -1</li>
</ul>
