<h1>Return the Sum of Two Numbers</h1>

<p>Create a function that takes two numbers as arguments and return their sum.</p>

<h2>Examples</h2>

<ul>
	<li>addition(3, 2) ➞ 5</li>
	<li>addition(-3, -6) ➞ -9</li>
	<li>addition(7, 3) ➞ 10</li>
</ul>
