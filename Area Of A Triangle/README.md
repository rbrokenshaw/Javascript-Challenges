<h1>Area of a Triangle</h1>

<p>Write a function that takes the base and height of a triangle and return its area.</p>

<h2>Examples</h2>
<ul>
	<li>triArea(3, 2) ➞ 3</li>
	<li>triArea(7, 4) ➞ 14</li>
	<li>triArea(10, 10) ➞ 50</li>
</ul>
