<h1>Count the Arguments</h1>

<p>Create a function that returns the number of argument it was called with.</p>

<h2>Examples</h2>
<ul>
	<li>numArgs() ➞ 0</li>
	<li>numArgs("foo") ➞ 1</li>
	<li>numArgs("foo", "bar") ➞ 2</li>
	<li>numArgs(true, false) ➞ 2</li>
	<li>numArgs({}) ➞ 1</li>
</ul>
